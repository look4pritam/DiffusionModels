# Diffusion Models

## Concept
- Forward Diffusion
  - Add (Gaussian) noise
  - Number of time steps
  - Markov chain - The future depends only on the present state, and not on the past history.
- Reverse Diffusion
  - Start with random noise
  - Remove noise
  - Reconstruct clear image
